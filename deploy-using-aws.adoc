= Use AWS to deploy control plane and Linux worker for NetApp Data Migrator
:icons: font
:imagesdir: ./media/

[.lead]
Use AWS to deploy the control plane VM and Linux worker VM for NetApp Data Migrator using Amazon Machine Images (AMIs). The Linux worker supports NFS data migrations. 

.Before you begin
Contact the NetApp Data Migrator team at `ndm-support@netapp.com` to request the AWS AMIs. Include the Account ID and region preference in the email or share it with your NetApp contact so that NetApp can share the images to that account ID.

.About this task
After requesting access to the AMIs, you copy the AMIs into your AWS account. Copying the AMIs gives you full ownership, avoids dependency on the source account, and is required for cross-region use.

Repeat the steps below for the control plane and Linux worker Amazon Machine Images (AMIs). 

.Steps
 
. Use the AMI IDs provided to locate the AMIs in your AWS account under *EC2* > *AMIs* > *Private images*.
. Copy the AMIs into your account: 
.. Select the AMI, then select *Actions* > *Copy AMI* 
.. Choose the target region (if needed). 
.. Enable encryption and select your own KMS key. 
.. Wait for the AMI status to show *Available*.
