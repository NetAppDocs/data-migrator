= Learn about NetApp Data Migrator support for features, file servers, and protocols
:icons: font
:imagesdir: ./media/

[.lead]
NetApp Data Migrator supports the following features, files servers, and protocols.

== Supported and unsupported features for NFS and SMB migrations

NetApp Data Migrator supports certain features for NFS and SMB migrations. Some features are not supported. Review the following tables to understand which features are supported and unsupported.

*Supported features*

[cols="1,2a" options="header"]
|===
| Description | Supported feature

| Permissions and audit handling
a| * Chain of custody reporting 
* Discretionary Access Control List (DACL)
* Preserve Access Control List (ACL)
* User Identifier (UID) and Security Identifier (SID) remapping 

| File system objects and links 
a|* Hard link handling 
+
NetApp Data Migrator migrates objects and links as separate files. 
* Symbolic links
+
*Note*: Supported for NFS migrations
| File metadata preservation 

a| * Preserve access time
* Preserve creation time
* Preserve modified time
* Preserve permissions

| Migration behavior and data consistency 
a| * Delete propagation 
* Incremental migration 
* Support for open files 
* Switchover support 

| Migration control and execution 
a| * File pattern exclusions
* Scheduling for migration
|===

*Unsupported features*

[cols="1,2a" options="header"]
|===
|Description | Unsupported feature

| Permissions and audit handling
| System Access Control List (SACL)

*Note*: Unsupported for SMB migrations
| File system objects and links 
a|* Follow NTFS junctions 
* Symbolic links
+
*Note*: Unsupported for SMB migrations

* Alternate data streams
+  
NetApp Data Migrator can discover but not migrate alternate data streams.
| File metadata preservation 
| Selective file attributes
| Migration behavior and data consistency 
| Migration of snapshots 
|===

== Supported file servers

NetApp Data Migrator supports the following file servers as source and destination for data migration.

[cols="1,2a" options="header"]
|===
| Description | Supported file server

| Source	a| Any NAS server, for example, Dell Isilon, ONTAP, Vanilla Linux, Windows, Cloud Volumes ONTAP
| Destination a| All service levels of Azure NetApp Files (ANF), Google Cloud NetApp Volumes (GCNV), Amazon FSx for NetApp ONTAP (FSxN), Cloud Volumes ONTAP
|===

== Supported NFS and SMB migration protocols

NetApp Data Migrator supports the following NFS and SMB protocol versions for data migration.

[cols="1,2a" options="header"]
|===
| Protocol | Supported versions

| NFS | 4.2, 4.1, 3.0
| SMB | 3.1, 3.0, 2.0 

|===
