= Learn about the NetApp Data Migrator protocols and file server support matrix
:icons: font
:imagesdir: ./media/

[.lead]
Learn about the protocols and file server types supported by NetApp Data Migrator to help streamline your migration activities.


|===
| Category | Supported protocols and file servers

| Source file server	 | Dell Isilon, ONTAP, Vanilla Linux, Windows, and Cloud Volumes ONTAP servers
| Destination file server | All service levels of Azure NetApp Files (ANF), Google Cloud NetApp Volumes (GCNV), Amazon FSx for NetApp ONTAP (FSxN), and Cloud Volumes ONTAP
| NFS protocol version| 3.0, 4.1
| SMB protocol version| 2.0, 3.0, 3.1
