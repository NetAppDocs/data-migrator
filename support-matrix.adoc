== Supported file servers

NetApp Data Migrator supports the following file servers as source and destination for data migration.

[cols="1,2a" options="header"]
|===
| File server | Description

| Source	a| Any NAS server, for example, Dell Isilon, ONTAP, Vanilla Linux, Windows, Cloud Volumes ONTAP
| Destination a| All service levels of Azure NetApp Files (ANF), Google Cloud NetApp Volumes (GCNV), Amazon FSx for NetApp ONTAP (FSxN), Cloud Volumes ONTAP
|===

== Supported NFS and SMB migration protocols

NetApp Data Migrator supports the following NFS and SMB protocol versions for data migration.

[cols="1,2a" options="header"]
|===
| Protocol | Supported versions

| NFS | 4.2, 4.1, 3.0
| SMB | 3.1, 3.0, 2.0 

|===
